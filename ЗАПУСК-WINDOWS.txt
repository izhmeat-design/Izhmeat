ЗАПУСК ЛОКАЛЬНОЙ ВЕРСИИ НА WINDOWS

1. Полностью распакуйте ZIP-архив в обычную папку, например на Рабочий стол.
   Не запускайте файл прямо из ZIP-архива.

2. Запустите файл:
   START-WINDOWS.bat

3. Если Node.js не установлен, файл сам скачает portable-версию Node.js в папку проекта:
   .tools/node
   Права администратора не нужны.

4. После этого скрипт сам:
   - создаст файл .env;
   - проверит зависимости;
   - запустит локальный сервер;
   - откроет сайт в браузере.

Адрес магазина:
http://localhost:3000

Админка:
http://localhost:3000/admin

Пароль администратора:
admin123

Чтобы остановить сервер, нажмите Ctrl+C в чёрном окне, затем Y и Enter.

Если автоматическая загрузка Node.js не пройдет, установите Node.js LTS вручную с https://nodejs.org/ и снова запустите START-WINDOWS.bat.
