# Лавка свежего мяса — GitHub Pages версия

Это статическая версия сайта для запуска только на GitHub Pages.

Админка: `admin.html`  
Инструкция: `README-GITHUB-PAGES-ADMIN.md`

В этой версии администратор каждый раз вводит GitHub token вручную. Token не хранится в файлах сайта.



## Telegram через Cloudflare Worker

См. инструкцию `CLOUDFLARE-WORKER-SETUP.md`.


## Обновление витрины магазина

В этой версии добавлены:

- крупные плитки категорий;
- блок «Товары дня»;
- блок «Какое мясо взять под блюдо»;
- блок «Как заказать»;
- мобильная нижняя панель: Категории / Каталог / Корзина / Позвонить;
- более компактные товарные карточки на мобильных устройствах.

«Товары дня» берутся из товаров с отметкой `popular: true`. Эту отметку можно менять в админке в разделе **Товары** через чекбокс **Популярный товар**.

Категории, подсказки и шаги заказа можно редактировать в `data/site.json` через JSON-редактор в админке.


## Исправление мобильной версии

Раздел **«Выбор мясника»** на мобильных устройствах переведён в горизонтальные компактные карточки:
- фото сверху;
- текст снизу;
- меньше высота блока;
- удобная прокрутка свайпом.


## Исправление фото в мобильном разделе «Выбор мясника»

Фото персонажей теперь отображаются целиком:
- головы не срезаются;
- используется `object-fit: contain`;
- фото выравниваются сверху;
- область изображения увеличена на мобильных экранах.


## Конструктор страницы в админке

В разделе **Редактирование сайта** добавлен блок **«Конструктор страницы»**.

Теперь можно редактировать сайт как мозаику:
- главный экран;
- категории;
- товары дня;
- выбор мясника;
- подбор мяса под блюдо;
- как заказать;
- каталог;
- доставка;
- описание магазина;
- контакты.

Для каждого блока можно менять:
- метку/подпись;
- заголовок;
- основной текст;
- фото;
- цветовую тему фона.

Доступные темы:
- стандартная;
- светлая;
- тёплая;
- кремовая;
- тёмная;
- фирменная красная.

Цветовые темы сохраняются в `data/site.json` в поле `sectionThemes`.

## Визуальный конструктор v4

В админке добавлен режим **Визуальный конструктор сайта**.

Где находится:
`Админка → Редактирование сайта → Визуальный конструктор сайта`

Как пользоваться:
1. В мини-версии сайта выберите нужный блок.
2. Нажмите **✎** — блок откроется в редакторе ниже.
3. Нажмите **+** — добавится новый дополнительный блок.
4. Нажмите **🗑** — обычный раздел будет скрыт, дополнительный блок будет удалён.
5. Измените текст, фото или цветовую тему и сохраните.

Важно:
- Встроенные основные разделы сайта не удаляются физически, а скрываются через `sectionVisibility`.
- Дополнительные блоки удаляются из `siteBlocks`.
- Цвета блоков сохраняются в `sectionThemes`.


## Визуальный конструктор v5 — перестановка блоков

В визуальном конструкторе добавлены кнопки:
- **↑** поднять блок выше;
- **↓** опустить блок ниже.

Порядок сохраняется в `data/site.json` в поле `sectionOrder`. После сохранения GitHub Pages обновит сайт, и блоки на главной странице будут отображаться в новом порядке.


## Визуальный конструктор v6 — редактирование карточек внутри блоков

При нажатии **✎** на блок теперь открывается:
1. общий редактор блока: заголовок, текст, фото, фон;
2. редактор внутренних карточек блока.

Можно редактировать без JSON:
- плитки категорий;
- карточки «Выбор мясника»;
- подсказки «Подбор под блюдо»;
- шаги «Как заказать»;
- карточки доставки;
- карточки главного экрана.

Для блока «Товары дня» карточки берутся из товаров с галочкой **Популярный товар**, поэтому они редактируются во вкладке **Товары**.
