КАК ЗАПУСТИТЬ ЛОКАЛЬНУЮ ВЕРСИЮ

WINDOWS
1. Распакуйте архив в обычную папку, например на Рабочий стол.
2. Откройте файл START-WINDOWS.bat двойным кликом.
3. При первом запуске файл сам установит зависимости.
4. Браузер откроется автоматически.

MACOS
1. Распакуйте архив.
2. Откройте START-MAC.command двойным кликом.
3. Если macOS не разрешит открыть файл, нажмите правой кнопкой мыши → Открыть.

LINUX
1. Распакуйте архив.
2. Откройте терминал в папке проекта и выполните ./START-LINUX.sh

АДРЕСА
Магазин:  http://localhost:3000
Админка:  http://localhost:3000/admin
Пароль:   admin123

ВАЖНО
Для запуска нужен Node.js 18 или новее.
Скачать: https://nodejs.org/

КАК ОСТАНОВИТЬ
В открывшемся окне нажмите Ctrl+C.
На Windows после вопроса подтверждения нажмите Y или Д и Enter.

ЕСЛИ ПОРТ 3000 ЗАНЯТ
Откройте файл .env и замените строку:
PORT=3000
например на:
PORT=3001
После этого сайт будет открываться по адресу http://localhost:3001
