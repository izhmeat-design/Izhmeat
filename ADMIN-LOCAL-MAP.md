# Карта локальной админки

Эта версия возвращена к локальной/серверной админке, где всё редактируется через привычные разделы.

## Как открыть

1. Запустите сайт локально.
2. Откройте:

```text
http://localhost:3000/admin
```

## 1. Раздел «Товары»

Здесь редактируются только товары каталога:

- название товара;
- категория;
- цена;
- единица измерения;
- описание;
- фото товара;
- услуги;
- КБЖУ;
- наличие;
- популярность.

Все товары сохраняются в:

```text
data/products.json
```

Фото товаров сохраняются в:

```text
public/uploads/
```

## 2. Раздел «Редактирование сайта»

Здесь редактируется сам сайт, кроме товаров.

### Основные разделы сайта

Здесь находятся:

- название сайта;
- телефон;
- email;
- слоган;
- главный заголовок;
- главный текст;
- главное фото;
- логотип;
- фото доставки;
- каталог;
- доставка;
- адреса;
- преимущества;
- способы оплаты;
- описание ассортимента.

Данные сохраняются в:

```text
data/site.json
```

Фото сохраняются в:

```text
public/uploads/
```

### Разделы со скриншота

Это отдельный понятный блок для тех участков, которые были на скриншотах:

- верхние маленькие карточки над главным заголовком;
- заголовок и описание блока «Свежее мясо»;
- большие карточки с фото, бейджем, заголовком и текстом.

### Карточки, статьи и блоки

Здесь можно добавлять, менять и удалять повторяющиеся элементы сайта:

- карточки главного экрана;
- карточки раздела «Свежее мясо»;
- услуги доставки;
- дополнительные статьи и блоки.

### Загрузка фото

Быстрая загрузка фото в папку:

```text
public/uploads/
```

После загрузки админка показывает путь к фото. Этот путь можно вставлять в блоки сайта.

### Полный JSON-редактор

Для расширенного редактирования всего сайта используется `site.json`.

Если непонятно, где находится конкретный текст или фото, можно найти его в JSON-редакторе через поиск браузера.

## Уведомления

В этой локальной версии уведомления Telegram/email находятся в разделе:

```text
Редактирование сайта → Уведомления о заказах
```

Telegram token и Chat ID берутся из `.env`.

## Важно про GitHub Pages

GitHub Pages-версия была упрощена из-за отсутствия сервера. Эта версия — локальная/серверная, поэтому в ней возвращены:

- привычная админка;
- уведомления Telegram;
- backend-заказы;
- сохранение заказов;
- полноценное редактирование сайта.
